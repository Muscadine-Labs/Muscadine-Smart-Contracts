# ERC20FeeSplitter (V1)

Fully immutable fee splitter contract for ERC20 tokens.

## Features

- Fully immutable - No owner, no configuration changes
- 50/50 split between two payees
- Deflationary token support
- Reentrancy protected

## Deployment

```bash
npm run deploy:base
```

## Testing

```bash
npm test
```
